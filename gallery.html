My Dress Up (990 X 1464)

I wanted to create a dress up game based off of me. The idea I came up with is based off of the simulation interface concept. I was inspired by a lot of dress up games from my childhood. I used that inspiration to create something that simulated my every day. In this project, I included Procreate sketches of items from my closet in an attempt to figure out something to wear. I used the button library for the main mechanics of this game. By using the buttons to navigate the pages as well as toggle the clothing items, it is easy for users to understand. Alongside the button library, I used the sound library to add background music as well as button clicks every time, well, a button is clicked. I used scenemanager to create the different pages.

A lot of my works are personal to me and this one is no exception. Enjoy helping me figure out what to wear :)

Link and Friends (800 x 700)

I based this work around the main character from the Legend of Zelda named Link. I worked in a group with members Cassandra Santander, Monzerrat Gomez, and Trang Nguyen. I mapped out their own custom arrays into my project alongside the ones I created.

I created 2 distinct arrays, one using color and shape and the other using text. I created 6 more functions for this array that are different from the original. This includes two functions utilizing images. One remapping is a Dark Link version of the original. Another is a Link in which the color is randomized. Lastly, one is a Link made of circles. For the text array, I created another function in which the grid size is different. I mapped out each function twice creating a total of 16 mapped images.

Untitled (600 X 600)

For this work, I created various ‘creatures’ based on the classic arcade game Pacman. First I created custom functions for Pacman’s body. I then created a bow function to create Mrs. Pacman. I then created child functions for the Ghosts of the game. Then in each parent function, I changed the color of the bodies. I created a vulnerable ghost (what appears after Pacman eats a ghost) based on the same functions for the normal ones. I did add a new set of functions for its eyes and mouth. Additionally to the characters, I created functions for a cherry object and the pellet that is in the original game. I added an image to the background to replicate the original game. After that, I created different arguments which moved my characters around.

Untitled 2 (600 x 600)

For this work, I created a class based on the red ghost I created from Untitled. Within the ghost class, I created three different methods. One changes the speed of the ghosts, the other changes the size, and the last changes the rotation. I made an array based on the ghost class and set the number to 15. I added a mousepressed function which triggers the various methods in the class.

Burnout (1920 x 1080)

Burnout is my favorite video that I created. It is about how draining it could be to a student and how they struggle to stay focused. Students would rather do anything else than pay attention. This is because they are experiencing burnout. Even when the person in the video was trying to study, they couldn't comprehend anything they were reading. Only when this student goes outside, away from the draining environment of the library, does the tone of the video change. They feel at peace and as if they were free. I myself was experiencing burnout during the filming of this video, which I hinted to at the end. This video was filmed with my phone and edited together uding Adobe Premiere Pro. I recorded the sounds with Voice Memos then edited it into a cohesive soundtrack.
